Summary of the method:

  1. Determine the number of input variables to know the size of the tables.
  2. Determine the number of output variables to define the number of tables to perform.
  3. Assign to the different products of the non-simplified equation a case of the table respecting the Gray code.
  4. Introduce the logical function into the table by positioning a '1' in the cases that validate the function of the output.
  5. Perform the grouping of adjacent cases.
  6. Output the simplified function by eliminating the input variables that change state.
