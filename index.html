Simplification Method: The Karnaugh Map

I. Introduction:

In a previous lesson (on Boolean logic), we learned that the method of simplifying equations by performing successive factorizations became very time-consuming and tedious as the number of variables increased. The Karnaugh map method allows us to perform simplifications much more quickly without having to write long equations.

II. The Karnaugh Map:

  • It is a table of 2n cases, n being the number of variables.
  • On the lines and columns, we place the state of the input variables in reflected binary code (Gray code).
  • In each of the cases, we place the state of the output for the corresponding input combinations.

In the example below, the number of variables is 4 because the Karnaugh Map contains 24 = 16 cases.

III. Simplification of equations from the Karnaugh Map:

The method consists of graphically highlighting, by a graphic process, the terms of a logic function that differ only by the state of a single variable (terms adjacent to each other). If the function logic depends on N variables, there are 2N possible cases (principle of the truth table).

Summary of the method:

  1. Determine the number of input variables to know the size of the tables.
  2. Determine the number of output variables to define the number of tables to perform.
  3. Assign to the different products of the non-simplified equation a case of the table respecting the Gray code.
  4. Introduce the logical function into the table by positioning a '1' in the cases that validate the function of the output.
  5. Perform the grouping of adjacent cases.
  6. Output the simplified function by eliminating the input variables that change state.
